# Objetivo da Solução

Nosso objetivo é proporcionar um espaço onde qualquer pessoa possa:

-Aprender novas habilidades com autonomia;
-Se preparar para o mercado de trabalho em constante evolução;
-Realizar reskilling e upskilling de forma prática;
-Acessar conteúdos educacionais de maneira inclusiva, acessível e flexível.
